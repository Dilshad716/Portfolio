Project Overview

Tic Tac Toe 2 is a classic game reimagined with power-based moves and strategic twists. In this version, strategy goes beyond Xs and Os. Each player has multiple moves, each with a unique power level. You can overwrite your opponent's move - but only if your move has a higher power.

Built as a casual/hyper-casual game inspired by classic Tic-Tac-Toe with new modes. The project features multiplayer functionality using Netcode for GameObjects, integrated Relay and Lobby systems, and implements design patterns like Singleton and Observer for clean architecture. The game logic is powered by 2D arrays with helper methods for efficient development.

Key Features

  • Power-based moves with strategic depth
  • Multiplayer functionality with Netcode
  • Relay and Lobby system integration
  • Singleton and Observer patterns
  • 2D array game logic
  • Helper methods for efficient development
  • Multiple game modes
