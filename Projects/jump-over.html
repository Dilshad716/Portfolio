Project Overview

Jump Over is a 2D rage-style climbing game where a lone stranger enters a magical world through a hidden portal. As he explores the enchanted realm, he encounters an ancient monk who offers a way home - but only if the stranger proves himself worthy by overcoming trials and mastering new skills.

The game features 2D monochromatic and silhouette art styles with interactive wind and fog shaders built using Shader Graph. I implemented smooth 2D movement and interaction scripts while optimizing mobile performance by faking depth of field effects.

Key Features

  • 2D rage-style climbing mechanics
  • Interactive wind shaders using Shader Graph
  • Fog shaders with Shader Graph
  • Smooth 2D movement & interaction
  • Mobile performance optimization
  • Faked depth of field effects
  • Monochromatic and silhouette art styles
